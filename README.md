# Hatio Task
The application is a github repo bookmark manager where the user can search the public github repos by name and user name and can select the specfic repos from the search result and can add to their bookmark list.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
